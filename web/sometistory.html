1] 배열과 링크  리스트의 차이점은?

배열 링크리스트
정적자료구조 동적자료구조
해당크기만큼의 연속된 메모리 주소를 할당 크기를 정할 필요가 없다.
인덱스값을 가진다. 노드가 존재하며 노드 안에 데이터가 있다.
접근과 탐색에 용이하다. 크기의 제한이 없다. 데이터 추가 삭제가 자유롭다.
크기를  미리 정해놓았기 때문에 수정하는 것이 불가능하다. 메모리주소를 할당 받지 않는다.
해당 배열 크기 이상의 데이터를 저장할 수 없다. 임의로 접근한것이 불가능하다. 데이터 탐색 시 순차적으로 접근해야한다.

[2] 상속과 다형성(Polymorphism)의 차이점은?
상속- 부모클래스와 자식 클래스 생성
다형성- 하나의 인터페이스나 부모클래스를 통해 여러 가지 구현을 할 수 있게 해주는 것
           - 하나의 객체가 여러가지 타입을 가질 수 있는 것.

[5] 서브쿼리와 조인의 차이점은?
- 두 개 이상의 테이블의 데이터를 추출하고 싶을 때 사용

서브쿼리 조인
여러 개의 테이블의 데이터를 추출할 때 사용 여러 개의 쿼리를 필요하지 않는다.(보통 2개정도)
쿼리 내에서 다른 쿼리를 실행하고 싶을 때  두개 이상의 테이블을 결합
가독성이 좋다  
 WHERE 절, HAVING 절, SELECT절 INNER JOIN, LEFT JOIN, RIGHT JOIN
